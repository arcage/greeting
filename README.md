# Greeting

This is just a sample repository for "Crystalの本2（仮）".

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  greeting:
    github: arcage/greeting
```

## Usage

```crystal
require "greeting"

Greeting.hello("John") #=> "Hello, John!"
```
